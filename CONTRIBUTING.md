# Contributing Guide

Thank you for wanting to contribute to the RXPS Wiki!

## Getting Started

### Step 1: Fork the Repository

1. Click the **Fork** button at the top right of this page
2. This creates your own copy of the repo

### Step 2: Create Your Content

1. Navigate to the appropriate folder (`docs/bosses`, `docs/items`, etc.)
2. Create a new `.md` file using the naming convention: `lowercase-with-dashes.md`
3. Copy content from the appropriate template in `/templates`
4. Fill in the information

### Step 3: Submit a Pull Request

1. Commit your changes with a descriptive message
2. Click **Contribute** then **Open Pull Request**
3. Describe what you added/changed
4. Submit!

## File Naming

Use lowercase with dashes:
- ✅ `alchemical-hydra.md`
- ✅ `dragon-claws.md`
- ❌ `Alchemical Hydra.md`
- ❌ `dragon_claws.md`

## Markdown Basics

### Headers
```markdown
# Main Title (H1)
## Section (H2)
### Subsection (H3)
```

### Bold & Italic
```markdown
**bold text**
*italic text*
```

### Links
```markdown
[Link Text](/path/to/page)
[External Link](https://example.com)
```

### Tables
```markdown
| Header 1 | Header 2 |
|----------|----------|
| Cell 1   | Cell 2   |
```

### Tips, Warnings, Notes
```markdown
:::tip Title
Tip content here
:::

:::warning Title
Warning content here
:::

:::note Title
Note content here
:::
```

## Adding Images

1. Add images to `static/img/` in the appropriate subfolder
2. Reference in markdown:
```markdown
![Alt text](/img/folder/image.png)
```

## Quality Guidelines

- **Accuracy** - Verify all stats and information in-game
- **Clarity** - Write for players of all experience levels
- **Completeness** - Include all relevant information
- **Formatting** - Use tables for data, headers for organization
