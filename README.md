# RXPS Wiki Contributor Templates

Welcome! This repo contains templates for contributing to the [RXPS Wiki](https://rxps.wiki).

## How to Contribute

1. **Fork this repository**
2. **Create or edit pages** using the templates below
3. **Submit a Pull Request** with your changes
4. **Wait for review** - a maintainer will review and merge your contribution

## Folder Structure

```
docs/
├── bosses/          # Boss guides
├── items/           # Item information
├── skills/          # Skill training guides
├── minigames/       # Minigame guides
├── guides/          # General guides
├── raids/           # Raid guides
├── economy/         # Economy guides
└── upgrades/        # Enchanting/upgrade guides
```

## Templates

Check the `/templates` folder for example page formats:
- `boss-template.md` - For boss guides
- `item-template.md` - For item pages
- `skill-template.md` - For skill guides
- `general-template.md` - For other pages

## Writing Guidelines

- Use clear, concise language
- Include accurate stats and information
- Add tables for data when helpful
- Test your markdown formatting
